## What is Dart?

Dart is a client-optimized language developed by Google. It is the language behind Flutter.

## Hello World

```dart title="main.dart"
void main() {
  print('Hello, Dart!');
}
```

## Variables

Dart is strongly typed. You can declare variables with `var`, explicit types, or `final`/`const`.

```dart
var name = 'Flutter';
String greeting = 'Hello';
final int year = 2024;
const double pi = 3.14159;
```
